Tuan's code

Boxed!In! is a game in which the player has to maneuver his spaceship around the waves of asteroids boxing in order to survive. This features collision detection, score tracking, and an incremental wave speed for each new level.

Color Guess is a game where the user has to pick the color tile corresponding with the provided RGB value. This features game reset and two different difficulty levels.

This web app connects with the WeatherUnlocked API to retrieve the current windspeed, temperature, and cloud coverage for Warner Robins, Georgia. It uses this information to report whether or not the conditions are optimal to fly a kite.

This program runs simulations for the A-Star Algorithm used in video games. Given a map populated with obstacles along with a user-inputted start and end coordinate, it will search the map for the shortest pathway between two endpoints and color it in.

This website displays organizes photos in an appropriate grid and adjust what's being displayed based on the size of the window.

ToDo List allows one to create a list of item and gradually check them off as they are completed. This features a toggling trashcan to delete each item and a keyboard listener.

YelpCamp is a full stack website for outdoor enthusiasts to create, share, and comment on their favorite camping sites between each other. This features authentication/authorization, Restful architecture and uses MongoDB, NodeJS, ExpressJS.
