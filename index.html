PoolPros

PoolPros dynamically creates dealer cards from a JSON file and allows one to filter them based on service credentials. This feature CORS request to retrieve the JSON data, VanillaJS, CSS from scratch, mobile-first design, and pop up modules.

KiteRunner

KiteRunner connects with the WeatherUnlocked API to retrieve the current windspeed, temperature, and cloud coverage for Warner Robins, Georgia. It uses this data to report whether or not the conditions are optimal to fly a kite. View Icon Credits

YelpCamp

YelpCamp is a full stack website for outdoor enthusiasts to create, share, and browse their favorite camping sites. This features authentication/authorization, Restful architecture and uses MongoDB, NodeJS, ExpressJS.

Boxed!In

Boxed!In! is a game in which the player has to maneuver his spaceship around the waves of asteroids boxing it in on all four sides. This features collision detection, score tracking, and an incremental wave speed for each new level.

Pathfinder

Pathfinder runs simulations of the A-Star Algorithm used in video games. Given a map populated with obstacles and a user-inputted start/end coordinate, it will search the map for the shortest pathway between two endpoints and color it in.

ToDoList

ToDoList allows one to add new items to a list, delete them as they become obsolete, and gradually check them off as they are completed.

Color Guess

Color Guess is a game where the user has to pick the color tile corresponding with the provided RGB value. This features a game reset and two different difficulty levels.

Photo Collage

Photo Collage organizes onto an appropriate Bootstraps grid and adjust what's being displayed based on the window size

Farm Blog

Farm Blog allows is a daily journal that allows a user to create a new entry, timestamps each post, and displays a summary of all the posts on the main page. This implements all 7 requests under the RESTful architecture style.
